# Dashboard de Vendas – Excel

## Descrição do Projeto
Este projeto consiste na criação de um dashboard de vendas no Excel, com o objetivo de transformar dados brutos em informações visuais claras e relevantes, facilitando a análise do desempenho comercial e apoiando a tomada de decisões baseada em dados.

O dashboard apresenta uma visão geral das vendas, permitindo identificar resultados por região, produtos mais vendidos e a evolução das vendas ao longo do tempo.

---

## Objetivo
Fornecer uma visualização clara e organizada dos dados de vendas, permitindo responder perguntas como:
- Qual é o desempenho geral das vendas?
- Quais regiões apresentam maior volume de vendas?
- Quais produtos se destacam?
- Como as vendas evoluem ao longo do tempo?

---

## Estrutura do Arquivo
O arquivo Excel está organizado em três abas principais:

- **Base**: contém os dados brutos utilizados no projeto.  
- **Análise**: utilizada para tratamento dos dados e criação de tabelas dinâmicas.  
- **Dashboard de Vendas**: painel visual com gráficos e indicadores para análise.

Essa separação facilita a organização e a manutenção do projeto.

---

## Indicadores e Visualizações
O dashboard apresenta os seguintes elementos:
- Total de vendas
- Vendas por região
- Vendas por produto
- Evolução das vendas ao longo do tempo

Os gráficos foram escolhidos de acordo com o tipo de dado e o objetivo da análise, priorizando clareza e fácil interpretação.

---

## Ferramentas Utilizadas
- Microsoft Excel
- Tabelas
- Tabelas Dinâmicas
- Gráficos
- Segmentações de dados

---

## Como Utilizar
1. Faça o download do arquivo `Desafio_Vendas.xlsx`
2. Abra o arquivo no Microsoft Excel
3. Acesse a aba **Dashboard de Vendas**
4. Utilize os filtros disponíveis para explorar os dados
